koken-google-site-verification
=================
This plugin adds a Google Search Console site verification HTML tag to your Koken CMS image publishing website.
You can add the generated verification code instead of manually uploading the static verification HTML file.

## Usage
### Install
Download or clone this repository and copy everything to:
 ```
storage/plugins/niklas-teich-google-site-verification/
```
### Configuration
Simply copy and use the generated verification code.
This plugin creates and adds the required HTML meta tag with your provided verification code.
